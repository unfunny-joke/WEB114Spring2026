Project 7 -- DOM and JS

JavaScript Questions:

1. Why is textContent safer than innerHTML?

Answer: textContent only refers to the plain text not the code, so the code is untouched by the process

2. How are HTML elements connected to JavaScript variables using DOM selection?

Answer: We can use methods such as .getElementByID() and .querySelector() to connect HTML elements to JavaScript variables since each HTML element is an object under DOM.

3. What would happen to the text box with id="text" if I added this line to the code above it:

text.textContent = "Hello!"

Answer: The code will override the original text with "Hello".
